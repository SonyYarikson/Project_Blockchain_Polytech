## 2. Общая информация о проекте.
### 2.1 Название и цель проекта
Проект "Дополнительное профессиональное образование (ДПО)" направлен
на создание современной цифровой платформы, обеспечивающей доступ к
программам повышения квалификации и профессиональной переподготовки
для специалистов различных отраслей.
Основные цели:
- Предоставление качественного онлайн-образования с учетом требований
рынка труда.
- Автоматизация процессов обучения (регистрация, оплата, доступ к
материалам, аттестация).
- Обеспечение гибкости обучения с возможностью выбора индивидуальных
траекторий.
### 2.2. Целевая аудитория
- Специалисты, желающие повысить квалификацию или получить новую
профессию.
- Работодатели, заинтересованные в обучении сотрудников.
- Образовательные организации, предлагающие программы ДПО.
### 2.3. Функционал платформы
- Личный кабинет для слушателей и преподавателей.
- Каталог программ с фильтрами по направлениям, длительности, стоимости.
- Онлайн-обучение (лекции, вебинары, тесты, практические задания).
- Аналитика успеваемости и вовлеченности пользователей.
### 2.4. Технологии и реализация
- Frontend: React.js (адаптивный дизайн).
- Backend: Strapi 4.
- База данных: PostgreSQL.
### 2.5. Этапы реализации
1. Анализ рынка и проектирование.
2. Разработка MVP (минимальной версии продукта).
3. Тестирование и доработка.
4. Запуск пилотных программ.
